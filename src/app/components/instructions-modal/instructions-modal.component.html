Instruções de Configuração

Por favor, siga os passos abaixo para configurar o projeto corretamente:

  1. Clique em Ir para Configurações do Projeto para configurar as definições do projeto.
    • Na tela de configurações do projeto, ajuste as informações necessárias, como nome do projeto, caminho do back-end e caminho do front-end.
    • Em seguida, clique no botão Configurar BackEnv para configurar as definições do ambiente de back-end. Preencha os campos e clique em Salvar.
    • Depois, clique no botão Configurar FrontEnv para configurar as definições do ambiente de front-end. Preencha os campos e clique em Salvar.
    • Após configurar todas as definições do projeto, clique em Salvar Projeto.
  2. Clique em Ir para Criação de Classes para configurar as classes do projeto.
    • Na tela de criação de classes, insira o número de classes que deseja criar e clique em Gerar Campos.
    • Para cada classe, adicione suas configurações e atributos.
    • Clique em Salvar Configurações após ajustar as configurações de cada classe.
    • Para os atributos de cada classe, configure as informações e clique em Salvar Atributos.
  3. Após salvar as configurações de todas as classes e atributos, retorne à tela principal.
  4. Finalmente, clique no botão Baixar MainConfig.mm para gerar e baixar o arquivo de configuração final.
